```

   └─ 収量予測

【Step 3：データ証明（SafelyChain）】
   └─ 生態系データの暗号化
   └─ 二重計上の排除
   └─ トレーサビリティ

【Step 4：金融化（MABC）】
   └─ Carbon Equity（炭素株式）
   └─ Yield‑Linked Token（収量連動トークン）
   └─ Quality Premium Bond（品質債券）
   └─ PBPEクレジット（ΔCベース）
```
